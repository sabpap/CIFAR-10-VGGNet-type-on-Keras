# CIFAR-10-on-Keras
VCNNet-type implementation for image classification on Keras
(trained on CIFAR 10 dataset)

## Files included:
-Build-Train_Model.py : script for creating and training cnn

-imshow.py    : custom imshow function

-model.h5
, model.json :  saved model
